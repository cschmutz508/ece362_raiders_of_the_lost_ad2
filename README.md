# ECE36200 Raiders of the Lost AD2

---

## Overview
This project involves the design and implementation of a multifunction embedded system featuring signal generation, signal analysis, display interfacing, and data storage capabilities. The system is being developed with a custom PCB and modular firmware architecture.

---

## Objectives: 

An ability to use PWM to simulate a function generator.
An ability to use ADC to convert analog data into a digital reading that can be displayed on our TFT display..
An ability to use spi to display waves with TFT display.
An ability to use spi to an sd card for exporting data.
